## 安全检查机制

虽然统一缩放到50%，但仍保留安全检查：

```typescript
// 如果缩放后仍然过大（极端情况）
if (obstacle.displayHeight > maxSafeHeight) {
  const additionalScale = maxSafeHeight / obstacle.displayHeight;
  obstacle.setScale(globalScale * additionalScale);
}
```

**作用**：
- 防止极端大图片破坏布局
- 确保游戏始终可玩
- 100%向下兼容

## 技术细节

### 缩放顺序
```
1. 加载原始图片
2. 应用 globalScale = 0.5
3. 检查是否超过安全边界
4. 如果超过，进一步缩小
5. 保存最终 displayWidth/Height
```

### 比例保持
```typescript
// ✅ 正确：使用 setScale
obstacle.setScale(0.5);
// 保持原始宽高比

// ❌ 错误：使用 setDisplaySize
obstacle.setDisplaySize(width * 0.5, height * 0.5);
// 可能改变比例
```
